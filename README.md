## 📖 프로젝트 소개 (About Project)
**CookitBeat**는 음악의 박자에 맞춰 재료를 손질하고 요리를 완성하는 **[리듬 액션 / 타이쿤]** 게임입니다. </br>
자바의 객체지향 특징을 활용하여 다양한 리듬 패턴과 요리 시스템을 구현했습니다.

* **개발 기간:** 2025.10 ~ 2025.12
* **개발 환경:** Java JDK 17, IntelliJ IDEA, Swing
